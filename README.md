# chatgpt-gemini.com
Comparison of html website building capabilities of both models: ChatGPT and Gemini. Please see the readme for the prompt.

## Prompt used for both models:
Create a modern, fully responsive landing page for a fictional dental clinic called Smile Care Dental using only HTML, CSS, and vanilla JavaScript in a single HTML file with embedded <style> and <script> sections. Do not use external libraries, frameworks, fonts, or images; use inline SVG for all illustrations and icons.
Use a clean light blue and white color scheme with rounded corners, soft shadows, smooth animations, and a professional, welcoming design.
## Include:
Sticky header with logo, navigation (Home, About, Dentists, Schedule, Reviews, Contact), and a mobile menu.
Hero with headline, intro, two CTA buttons, and a dental-themed SVG illustration.
About section with three feature cards.
Dentists section with four profile cards (SVG avatar, name, specialty, experience).
Schedule section with office hours and an emergency notice.
Reviews section with three ★★★★★ testimonials.
Footer with contact information and SVG social icons.
Add smooth scrolling, fade-in animations using Intersection Observer, hover effects, a scrolling header effect, semantic HTML, accessibility best practices, and clean, well-commented code.
